@startuml
title 金融管理系统操作流程

actor 用户 as User

== 用户注册操作流程 ==
User -> 注册页面 : 访问注册页面
注册页面 -> 浏览器 : 填写信息并提交
alt 注册成功
    浏览器 -> 注册页面 : 显示注册成功信息
    User -> 登录页面 : 访问登录页面
else 注册失败
    浏览器 -> 注册页面 : 返回注册页面并显示错误提示
end

== 用户登录操作流程 ==
User -> 登录页面 : 输入手机号和密码并提交
alt 登录成功
    登录页面 -> 主页界面 : 跳转到主界面
else 登录失败
    登录页面 -> 登录页面 : 显示错误信息
end

== 账户管理操作流程 ==
User -> 账户页面 : 进入账户页面
账户页面 -> 账户详细信息 : 显示用户信息
账户页面 -> 转账页面 : 点击转账按钮
账户页面 -> 黄金花头像 : 点击查看黄金
账户页面 -> 股票页面 : 点击查看股票

== 资产管理操作流程 ==
User -> 资产页面 : 进入资产页面
资产页面 -> 资产处理 : 资产处理
资产页面 -> 总资产界面 : 显示总资产信息
资产页面 -> 更新资产信息 : 更新资产信息

== 转账处理流程 ==
User -> 转账页面 : 访问转账页面
转账页面 -> 转账处理 : 输入转账信息并提交
alt 余额不足
    转账页面 -> 提示 : 提示余额不足
else 余额足够
    转账页面 -> 提示 : 提示转账成功
end

== 黄金花管理操作流程 ==
User -> 黄金详细页面 : 访问黄金详细页面
黄金页面 -> 黄金购买页面 : 点击购买按钮
黄金购买页面 -> 黄金购买处理 : 输入购买信息并提交
alt 余额不足
    黄金页面 -> 提示 : 提示余额不足
else 余额足够
    黄金购买页面 -> 提示 : 购买成功
end

User -> 添加银行卡页面 : 点击添加银行卡按钮
添加银行卡页面 -> 添加银行卡处理 : 提交银行卡信息
添加银行卡页面 -> 提示 : 添加银行卡成功
黄金购买页面 -> 提示 : 返回成功信息

== 股票管理操作流程 ==
User -> 股票页面 : 查看股票详细信息
股票页面 -> 股票购买页面 : 点击购买股票
股票页面 -> 股票购买处理 : 输入购买信息并提交
alt 余额不足
    股票页面 -> 提示 : 提示余额不足
else 余额足够
    股票页面 -> 提示 : 购买成功
end

@enduml
